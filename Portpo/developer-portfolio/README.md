# Developer Portfolio

This is a simple developer portfolio website built using HTML5, CSS3, and JavaScript. It showcases the developer's skills, projects, and contact information.

## Project Structure

```
developer-portfolio
├── index.html        # Main homepage of the portfolio
├── about.html        # Information about the developer
├── projects.html     # Showcase of projects
├── contact.html      # Contact form and information
├── css
│   ├── style.css     # Main styles for the website
│   └── responsive.css # Responsive styles for various devices
├── js
│   ├── script.js     # Main functionality and interactivity
│   └── utils.js      # Utility functions for reuse
├── assets
│   └── fonts         # Directory for font files
├── README.md         # Project documentation
└── package.json      # Configuration file for npm
```

## Features

- Responsive design that adapts to different screen sizes.
- Interactive elements powered by JavaScript.
- Clear sections for about, projects, and contact information.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd developer-portfolio
   ```
3. Open `index.html` in your web browser to view the portfolio.
